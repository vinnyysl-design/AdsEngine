# MelieAds React

Versão React/Vite do dashboard MelieAds, pronta para GitHub e publicação no Netlify.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

A pasta gerada será `dist/`.

## Publicar no Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## Observação

Esta versão é uma adaptação visual e funcional para web em React/Vite. Ela mantém o conceito do dashboard original, com KPIs, gráficos, recomendações e importação simples de Excel no navegador.
